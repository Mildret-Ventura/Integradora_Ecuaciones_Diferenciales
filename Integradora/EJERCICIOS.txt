    GUÍA DE EJERCICIOS Y EXTENSIONES PRO - LAPLACE STUDIO
========================================================================

Este documento detalla los 4 casos mínimos requeridos por la tarea 
integradora y las extensiones opcionales implementadas.

--- 1. CASOS DE PRUEBA MÍNIMOS ---
Caso 1: y' + 2y = 0, y(0) = 1  => y(t) = exp(-2t)
Caso 2: y'' + 3y' + 2y = 0, y(0) = 0, y'(0) = 1  => y(t) = exp(-t) - exp(-2t)
Caso 3: y' - y = exp(t), y(0) = 1  => y(t) = (t + 1)*exp(t)
Caso 4: y'' + y = sin(t), y(0) = 0, y'(0) = 0  => y(t) = sin(t)/2 - t*cos(t)/2

--- 2. EXTENSIONES OPCIONALES IMPLEMENTADAS ---
A. Interfaz Gráfica Moderna: Usando CustomTkinter con estética Dashboard.
B. Interpretador de Texto: Acepta notación natural (y'', y', etc.) 
   mediante el uso de expresiones regulares.
C. Exportación a PDF: Botón de generación automática de reportes.
D. Comparación con Métodos Numéricos: Casilla de verificación para 
   visualizar la comparación en la gráfica.

   condiciones iniciales en las derivadas. SymPy es potente, pero 
   requiere un tratamiento cuidadoso de los términos para que la 
   resolución algebraica sea exitosa.
